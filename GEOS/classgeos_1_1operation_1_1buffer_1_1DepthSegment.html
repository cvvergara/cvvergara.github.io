int geos::operation::buffer::DepthSegment::compareTo ( const DepthSegment &  other) const
inline

Defines a comparision operation on DepthSegments which orders them left to right

DS1 < DS2   if   DS1.seg is left of DS2.seg
DS1 > DS2   if   DS1.seg is right of DS2.seg
Parameters
obj
Returns

try and compute a determinate orientation for the segments. Test returns 1 if other is left of this (i.e. this > other)

If comparison between this and other is indeterminate, try the opposite call order. orientationIndex value is 1 if this is left of other, so have to flip sign to get proper comparison value of -1 if this is leftmost
