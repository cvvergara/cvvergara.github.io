QuadEdge & geos::triangulate::IncrementalDelaunayTriangulator::insertSite ( const quadedge::Vertex &  v)

Inserts a new point into a subdivision representing a Delaunay triangulation, and fixes the affected edges so that the result is still a Delaunay triangulation.

Returns
a quadedge containing the inserted vertex

This code is based on Guibas and Stolfi (1985), with minor modifications and a bug fix from Dani Lischinski (Graphic Gems 1993). (The modification I believe is the test for the inserted site falling exactly on an existing edge. Without this test zero-width triangles have been observed to be created)

Connect the new point to the vertices of the containing triangle (or quadrilateral, if the new point fell on an existing edge.)
